Morphogenetic Audio-Reactive Particle System

This web platform is an experimental, long-term research project focused on real-time audiovisual synthesis in the browser. At its core, the application explores the intersection between computational physics, procedural geometry, and sound-driven interaction.

The system renders and simulates hundreds of thousands of particles entirely on the GPU. Instead of calculating motion on the CPU, all particle dynamics are computed inside fragment shaders, allowing high-density simulations (up to 512×512 particle grids) to run smoothly in real time.

How It Works (High-Level Overview)

The application is built around three core components:

1. GPU-Based Particle Simulation

Each particle exists as data stored in textures. Position, velocity, and other physical parameters are updated every frame directly on the GPU. This avoids CPU bottlenecks and enables large-scale parallel computation.

2. Physics-Driven Motion

Particles are influenced by a set of mathematically defined forces, including:

  • Harmonic attraction fields
  • Damping forces
  • Vortex fields
  • Toroidal implicit constraints
  • Divergence-free curl noise

These forces are combined to produce emergent, organic motion patterns. Geometry is not explicitly modeled; instead, it emerges from implicit constraints applied to particle behavior.

3. Audio-Reactive Parameter Mapping

Audio input does not directly move particles. Instead, it modulates physical parameters of the system. Frequency bands and amplitude envelopes are mapped to force intensities, damping coefficients, and field distortions. This results in structured, physically coherent transformations rather than arbitrary visual reactions.

Technical Philosophy

The project prioritizes:

  • Full GPU parallelization
  • Deterministic, mathematically coherent force systems
  • Emergent geometry rather than predefined models
  • Clean parameter mapping between sound and physics
  • Scalability for high particle counts and multi-user access

The architecture is designed to remain modular, allowing future integration of additional force models, new audio mapping strategies, and alternative rendering techniques.

Project Status

This is a private, independent, long-term project currently in continuous development. The system evolves incrementally, with ongoing refinements to performance, physical modeling, visual fidelity, and audio responsiveness.

New features, optimizations, and structural changes are introduced progressively as part of an iterative research process.
